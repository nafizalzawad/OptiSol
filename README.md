# OptiSol

OptiSol is an AI-driven solar panel cooling and efficiency optimization system designed for hot climates.

## About

OptiSol monitors panel temperature, predicts overheating using AI, and activates targeted fan and micro-mist cooling to increase energy output while minimizing water and power consumption.

## Features

- **Real-time IoT Monitoring**: Sensors stream panel, ambient, and flow metrics.
- **AI Thermal Prediction**: Forecasts overheating early.
- **Smart Cooling Control**: Targeted fans and micro-mist cooling.
- **Water Optimization**: Closed-loop scheduling to minimal waste.
- **Dashboard Analytics**: View temperature, gain, water use, and alerts.

## Technology Stack

- **Frontend**: React, TypeScript, Vite
- **UI Framework**: shadcn/ui, Tailwind CSS
- **Visualization**: Recharts

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```
